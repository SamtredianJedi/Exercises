Hi! At times, beginners always find it hard getting the application of the theory they learn in programming or a particular language. In this article, we'll look at another application of JavaScript. We'll see the use/application of the switch statement which will help us build a simple web dictionary app. I called it a web dictionary app because the switch statement gives us the opportunity to program different outputs when different requests are made. The application will get user input via a form and will output the result as an alert. It equally has an assigned default value if no result is found. First of all, let's begin by revising the syntax of the switch statement. The switch statement is used to avoid writing several if...else statements. switch(expression) { case a: code block break; case b: code block break; default: code block } Therefore, it evaluates different entries(case) and checks for its specific output, else it displays the default value.
